# my_cafebook

## サイト概要
### サイトテーマ
カフェ好きのためのレビューサイト

### テーマを選んだ理由
私はカフェ巡りが好きなのですが、地域の個人店などのカフェはなかなか情報が出て来ずらかったりします。
そこでカフェ好きのレビューサイトをつくり、各地のカフェ情報を募りたいと考えました。

### ターゲットユーザ
カフェ好きで、様々なカフェに行ってみたい方。

### 主な利用シーン
お気に入りのカフェを見つけた時や、新しいカフェに行きたいと思った時。
